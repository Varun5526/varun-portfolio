# 🌟 Varun's Portfolio

* **Dynamic Animations:** Micro-interactions and reveals powered by `framer-motion`.

* **Project Showcases:** Interactive modals with deep-dives into metrics and tech stacks.

## 🛠️ Installation & Setup

1. **Clone the Repo:**

   ```bash
   git clone https://github.com/Varun5526/varun-portfolio.git
   ```

2. **Install Dependencies:**

   ```bash
   npm install
   ```

3. **Launch Dev Server:**

   ```bash
   npm run dev
   ```
